== 试一试！字符串SQL注入

代码中的查询构建了一个动态查询，如前面的示例所示。该查询通过连接字符串构建，容易受到字符串SQL注入的攻击：

------------------------------------------------------------
"SELECT * FROM user_data WHERE first_name = 'John' AND last_name = '" + lastName + "'";
------------------------------------------------------------

尝试使用下面的表单从用户表中检索所有用户。您不需要知道任何特定的用户名即可获取完整列表。
