== SQL注入的严重性

=== SQL注入攻击的严重性受限于以下因素：
* 攻击者的技能和想象力
* 深度防御对策
** 输入验证
** 最小权限原则
* 数据库技术

=== 并非所有数据库都支持命令链
* Microsoft Access
* MySQL Connector/J 和 C
* Oracle

=== SQL注入在PHP、Classic ASP、Cold Fusion和较旧语言中更常见
* 这些语言不提供参数化查询支持
* 新版本已添加参数化查询功能
* 这些是Web技术的早期采用者（即旧代码）

=== 并非所有数据库都相同（以SQL Server为例）
* 命令shell：`master.dbo.xp_cmdshell 'cmd.exe dir c:'`
* 注册表命令：`xp_regread`、`xp_regdeletekey`等
